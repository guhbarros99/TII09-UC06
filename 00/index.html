Introdução ao Ambiente de Desenvolvimento

Objetivos da Aula

Familiarizar os alunos com as ferramentas necessárias para o desenvolvimento de algoritmos em JavaScript, incluindo Visual Studio Code, entrada e saída de dados, uso do debugger, Git e GitHub.

Conteúdo da Aula

  1. Configuração do Ambiente de Desenvolvimento

    • Baixar e instalar o Visual Studio Code, caso não tenha.
    • Configurar extensões úteis para JavaScript, como:
      • GitLens (para facilitar o uso do Git).
      • vscode-icons
      • draw io
      • IntelliCode
      • Live Server
      • Live Preview
      • Debugger for Firefox
    • Ajustar configurações básicas (tema, logint do github e formatação automática).

    Criando o Primeiro Arquivo JavaScript

    • Criar uma pasta chamada algoritmos-javascript.
    • Abrir o VS Code e a pasta criada.
    • Criar um arquivo aula1.html.
    • Adicionar o seguinte código::
      JavaScript
      
<script>
console.log("Olá, mundo! Este é meu primeiro código em JavaScript.");
</script>
      • Abrir o Console do Navegador (F12 > Console).
      • Verificar mensagem.

  2. Entrada e Saída de Dados no Console e Browser

    • Saída de Dados com console.log()

      • O console.log() permite exibir mensagens e valores no console do navegador:
        JavaScript
        
console.log("Este é um exemplo de saída de dados no console.");
let numero = 10;
console.log("O valor da variável número é:", numero);

    • Entrada de Dados com prompt()

      • O prompt() permite capturar dados do usuário via navegador:
        JavaScript
        
let nome = prompt("Qual é o seu nome?");
console.log("Bem-vindo, " + nome + "!");
      • Importante: Sempre executar esse código direto no navegador (ou em um arquivo HTML contendo <script src="aula1.js"></script> quando necessário).
    • Exemplo Prático: Somando Dois Números
      JavaScript
      
let num1 = prompt("Digite o primeiro número:");
let num2 = prompt("Digite o segundo número:");
let soma = Number(num1) + Number(num2);
console.log("A soma dos números é:", soma);
    • Conversão de Tipos: Como prompt() retorna uma string, usamos Number() para garantir que a soma funcione corretamente.

  3. Uso do Debugger no VS Code

    • Debugando Código JavaScript
      1. Criar um novo arquivo debug.js e adicionar o código:
        JavaScript
        
let a = 5;
let b = 10;
let resultado = a + b;
console.log("Resultado da soma:", resultado);
      2. No VS Code:
        • Colocar um breakpoint: Clique na lateral esquerda do número da linha onde deseja parar.
        • Abrir a aba "Run & Debug" (Ctrl + Shift + D).
        • Executar com "Start Debugging" (F5).
        • Observar valores das variáveis e fluxo de execução.
    • Uso da Palavra-chave debugger
      JavaScript
      
let x = 7;
let y = 3;
debugger;
let resultado = x * y;
console.log("Resultado da multiplicação:", resultado);

  4. Configuração Inicial do Git

    1. Configurar nome e e-mail

      GitBhash
      
git config --global user.name "Seu Nome"
git config --global user.email "seuemail@example.com"

    2. Criando um Repositório Git

      GitBhash
      
git init
git add .
git commit -m "Primeiro commit - configuração do projeto"

    3. Publicando no GitHub

      GitBhash
      
git remote add origin https://github.com/seuusuario/algoritmos-javascript.git
git push -u origin main
