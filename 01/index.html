Conceitos Fundamentais de Algoritmos e Lógica de Programação

Objetivos da Aula

Introduzir os conceitos de algoritmos, pensamento computacional, pseudocódigo e fluxogramas, preparando os alunos para a construção de soluções lógicas eficientes.

Conteúdo da Aula

  1. O que é um Algoritmo?

    1. Definição.

      Um algoritmo é uma sequência finita e ordenada de passos para resolver um problema. Ele pode ser descrito em linguagem natural, pseudocódigo, fluxograma ou código de programação.

    2. Exemplos de Algoritmos no Dia a Dia
      • Receita de bolo (passos para preparar um bolo).
      • Rotina matinal (levantar, escovar os dentes, tomar café, sair de casa).
      • Processos industriais (montagem de um carro).

  2. Pensamento Computacional

    1. Definição e Componentes
      • Decomposição (dividir o problema em partes menores).
      • Reconhecimento de padrões (identificar semelhanças e tendências).
      • Abstração (focar no que é essencial e ignorar detalhes desnecessários).
      • Algoritmos (criar uma sequência de passos para resolver um problema).

    2. Atividade Prática

      Resolver o problema de calcular a média de um aluno.

      • Decomposição: Precisamos de notas, um cálculo de média e um critério de aprovação.
      • Padrões: O cálculo da média segue sempre a mesma estrutura (somar e dividir).
      • Abstração: Ignorar detalhes como o tipo da prova, o nome do aluno, etc.
      • Algoritmo: Criar os passos do cálculo da média (será visto em pseudocódigo na próxima seção).

  3. Representação de Algoritmos

    1. Linguagem Natural

      Antes de escrever código, podemos descrever um algoritmo com palavras. Exemplo: Algoritmo para somar dois números e mostrar o resultado.

      1. Pedir o primeiro número.
      2. Pedir o segundo número.
      3. Somar os dois números.
      4. Mostrar o resultado.

    2. Pseudocódigo

      Pseudocódigo é uma forma estruturada de representar algoritmos usando uma linguagem próxima da humana, mas com estrutura lógica semelhante à da programação.

      Sintaxe básica de pseudocódigo:

      • Entrada de dados: LER
      • Processamento: CALCULAR
      • Saída de dados: EXIBIR

      • Exemplo:

        Pseudocodigo
        
INICIO
    ESCREVER "Digite o primeiro número:"
    LER num1
    ESCREVER "Digite o segundo número:"
    LER num2
    soma ← num1 + num2
    ESCREVER "A soma é:", soma
FIM

  4. Fluxogramas

    O que é um Fluxograma?

    Fluxogramas são representações gráficas de algoritmos, utilizando símbolos padronizados:

    • Início/Fim: Oval
    • Processo: Retângulo
    • Decisão: Losango
    • Entrada/Saída: Paralelogramo

Exercício

Exercícios Práticos

  1. Escreva um pseudocódigo para calcular a área de um círculo.
  2. Desenhe um fluxograma para determinar se um número é par ou ímpar.
  3. Implemente no JavaScript um algoritmo que peça a idade do usuário e exiba se ele é maior de idade ou não.
