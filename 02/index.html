Variáveis, Constantes e Tipos de Dados

Objetivos da Aula

Ensinar como declarar e manipular variáveis e constantes, entender tipos de dados, realizar operações aritméticas e lógicas, além de explorar boas práticas e conversões de tipo no JavaScript.

Conteúdo da Aula

  1. Introdução a Variáveis e Constantes

    1. O que são Variáveis?

      Variáveis armazenam valores que podem ser alterados durante a execução do programa. No JavaScript, usamos:

      • let → permite reatribuição do valor.
      • var → desatualizado, evita-se o uso.

      Exemplo:

      JavaScript
      
let nome = "Ana";
let idade = 25;

    2. O que são Constantes?

      Constantes armazenam valores fixos, ou seja, não podem ser alterados. No JavaScript, usamos const.

      Exemplo:

      JavaScript
      
const PI = 3.14159;
const anoNascimento = 1995;

  2. Tipos de Dados no JavaScript

    1. Principais Tipos de Dados
      2. Tipo Exemplo Descrição
      String "Olá, mundo!" Texto
      Number 25 Número
      Boolean true Verdadeiro/Falso
      Null null Nulo
      Undefined undefined Não definido

    2. Verificando o Tipo de um Dado

      Utilizamos typeof para descobrir o tipo de uma variável.

      Exemplo:

      JavaScript
      
let idade = 25;
console.log(typeof idade); // number

let nome = "Carlos";
console.log(typeof nome); // string

  3. Operadores Aritméticos e Lógicos

    1. Operadores Aritméticos

      Operador Descrição
      + Soma
      - Subtração
      * Multiplicação
      / Divisão
      % Módulo

      Exemplo:

      JavaScript
      
let num1 = Number(prompt("Digite o primeiro número:"));
let num2 = Number(prompt("Digite o segundo número:"));

console.log("Soma:", num1 + num2);
console.log("Diferença:", num1 - num2);
console.log("Produto:", num1 * num2);
console.log("Divisão:", num1 / num2);
console.log("Resto da divisão:", num1 % num2);

    2. Operadores Lógicos

      Operador Descrição
      && E
      || Ou
      ! Não

  4. Conversão de Tipos

    1. Conversão Explícita

      Convertendo um valor para número:

      JavaScript
      
let numero = Number("10"); // Converte string "10" para número 10
let inteiro = parseInt("10.5"); // Converte string para número inteiro (10)
let decimal = parseFloat("10.5"); // Converte string para número decimal (10.5)

      Diferença entre Number e parseInt

      JavaScript
      
console.log(Number("10.5")); // 10.5
console.log(parseInt("10.5")); // 10
    2. Conversão de Dados do prompt
      3. JavaScript
      
let idade = prompt("Digite sua idade:");
console.log(typeof idade); // string

idade = Number(idade);
console.log(typeof idade); // number

Exercícios

Criação de Algoritmo e Fluxograma

  1. Peça ao usuário um número.
  2. Verifique se ele é par ou ímpar.
  3. Exiba o resultado no console.
  4. Faça o fluxograma desse processo.
  1. Peça ao usuário o peso (kg) e altura (m).
  2. Calcule o IMC usando a fórmula IMC = peso / (altura * altura).
  3. Exiba o resultado e classifique de acordo com a tabela:
    • IMC < 18.5 → Abaixo do peso
    • 18.5 ≤ IMC < 25 → Peso ideal
    • IMC ≥ 25 → Acima do peso
